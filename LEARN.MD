# LEARN.MD — Audit Knowledge Base

## SEO / Indexing
- **Learning**: Sitemap contains only 38 of 397 pages. ~342 word pages, 13 Hebrew category pages, 3 age-bracket pages, and 1 guide are missing. Sitemap is hand-maintained, not auto-generated from build output. This is the root cause of the indexing gap (only ~7 indexed as of Mar 14).
- **Signal**: Compare `wc -l public/sitemap.xml` against total pages generated by `generate-seo-pages.js`. Check GSC "Pages" report.
- **Action**: Auto-generate sitemap from build output. Add hreflang cross-references for Hebrew pages (`HREFLANG_HE` is currently empty string). Resubmit to GSC after fix.
- **First seen**: 2026-03-13
- **Updated**: 2026-03-14 — root cause identified (sitemap gap, not just indexing lag)
- **Confidence**: High

## Observability

- **Action**: Add `setTag('appVersion')`, set anonymous player ID as user context, reduce trace rate to 0.1.
- **First seen**: 2026-03-13
- **Updated**: 2026-03-14 — configuration gaps identified
- **Confidence**: High
